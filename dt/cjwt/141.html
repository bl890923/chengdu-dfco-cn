UI设计师需要熟知的平面基础

作者： 时间：2021-05-25 16:03:26

随着互联网的发展带动了 UI 设计的短暂刚需，经过 3-5 年的发展，UI 设计也开始趋于平缓推进中。在此期间很多设计专业内部转岗与非行业转行也是一度热潮。

其中平面设计、电商设计、网页设计等转 UI 设计算是设计内部转岗最便利的，由于已经具备软件基础与设计基础，相对较为容易。也有一部分是直接接触 UI 设计的，虽然可以快速入行，但是美术功底与设计基础相对还是有所欠缺。

最早之前我也是视觉传达艺术设计专业毕业，经历了平面、品牌设计、电商设计等，最后在 UI 萌芽时期加入到其中。今天特意为各位 UI 设计小伙伴梳理了一下平面相关的内容，希望加深大家的平面设计基础，为 UI 设计视觉层面深入化打下点滴基础。

 

分享思路

 

本文将从平面设计认知、平面设计相关的专业基础、印刷须知与对接事项、设计师的职业路径和平面设计对 UI 设计的影响几个维度进行分析。希望通过本文的梳理带给更多 UI 设计师对于平面设计基础的理解，加强平面设计功底将有助于专业深度的进阶。

 

一、认知

 

1、初识平面设计

 

1.1、关于

平面设计也称为视觉传达设计，是以“视觉”作为沟通和表现的方式。相对于 UI 设计来说平面设计要早得非常多，在现在的设计职场中平面设计负责的工作内容相对比较多，对于他们的设计功底也是要求比较全面的。

作为平面设计师会接触到包括标识（商标和品牌）、出版物（杂志，报纸和书籍等）、平面广告，海报，广告牌，网站图形元素、标志和产品包装等等方面的设计需求。一般做过三年以上平面的设计师，应该需要具备较好的图形创意、板式设计、视觉表现、品牌设计等方面的专业能力。很多平面设计师转行 UI 设计相对比较容易，就是他们具备了很好的设计功底。

那么作为 UI 设计师具备基本的平面设计功底是非常必要的，可以使得自己的设计更加深入稳定，也能将图形设计、插画设计、品牌思维等融入到产品 UI 设计中，加强视觉层面的说服力。

1.2、平面设计师专业广度

平面设计是沟通传播、风格化和通过文字和图像解决问题的艺术，作为专业的平面设计师需要具备更多的专业广度。除了软件技能操作以外，还需要具备信息处理与排版的能力、图形创意与品牌思维的能力、各种商业传播的视觉表现与处理、字体设计等等方面的能力。

所以，一个合格的平面设计师至少也得打磨三年以上，俗话说三年入行就是这个理。

1.3、平面设计师职业现状

其实从设计专业广度出发，平面设计师具备的设计能力和职场中解决设计问题的能力都是比较全面的。由于国内大部分的公司都处于发展期，对于设计师的认知还在逐步提升中，导致平面在公司里面需要完成的设计任务很多（无形中就被认为是杂事一堆啦）。

很多公司都只有一两个设计师，给到的设计周期又很少，导致作品基本是赶出来的，质量肯定是比较难以控制的。除非能够进入一些 4A 公司和具备成熟的设计团队的公司，不然其实平面的待遇还是相对较差的，也有逐步进入管理层的，相对还算可以的。从就业的角度来看，平面设计师找工作还是相对比较容易的。

1.4、不可小瞧的设计功底

平面设计师虽然平时工作内容相对比较杂，但是锻炼出来的设计功底还是非常不错的。虽然大部分设计师初入行业都可能去的是中小型企业，接触的内容比较杂，但是往往这样能够学到更多的职场技能。

工作就是能够解决更多的问题，中小企业的设计经历可以让我们学到很多，磨练自己的设计功底，为后续的精专方向打下基础。

 

2、平面设计师的进阶之路

2.1、杂学活用

作为平面设计师在职场中，需要不断地扩展自己的专业广度，因为你会遇到很多的设计杂事，之前做平面的时候连老板的 PPT 也是需要设计的 o(╥﹏╥)o 。在工作中需要多一份主动，闲余的时间可以思考一下公司的一些设计优化，将你想要的设计感觉逐步去推进，这样以后才能逐步的积累一些不错的设计方案。

2.2、精专细分领域

在平面设计中，也有设计师专攻一些设计板块，这样通常会做得特别出色。例如，有些设计师专门设计电影海报、书籍设计、品牌设计、包装设计等等。在一个阶段时间里面只有研究一个领域的设计，才能有更多精力和时间去进行深入的学习研究。

通过前期的综合性锻炼打下一定的设计基础，后面如果主攻一个领域的设计也是非常有利的，一切的经历都是专业的基础，从而厚积薄发。

2.3、扩展互联网设计能力

作为互联网时代下的设计师，需要顺势而发展，不断寻求突破、精进与转变。如果你想要改变和提升自己的专业能力，除了在细分领域进行深入研究以外，学习新的设计形式与趋势是必不可少的，这也是很多平面转 UI 设计的原因之一。

很多平面设计师经常会问，现在 UI 设计师已经很多了，这个时候转行加入会不会太迟了。其实这里大家需要思考的并非是转入做 UI 设计的问题，而是提升自己专业广度与匹配互联网时代下的设计需求的问题。平面设计师学习一定的互联网设计只会增加自己的专业竞争力，现在很多 UI 设计师不仅具备产品设计能力，也具备其它视觉设计能力，如运营设计、电商设计、品牌设计、插画设计和一些印刷品的设计等等。

相对来说，在以后的设计职场中，一些专业广度更大的设计师必将具备更强的设计竞争力。还有就是在细分领域具备很强专业能力的设计师也会继续走热，未来无论是什么类型的企业，都会更加注重质量、服务和用户体验。那么我们作为视觉层面，产品的感官体验也是其中需要提升的板块之一。

2.4、管理路线

也有不少设计师工作几年后就进入管理层，有纯管理路线和专业兼备的路线。一些专业能力较好的设计师由于后期团队发展壮大，逐步就带起新人设计师，自然而然地进入到管理团队中。

这里也给一些新入行业的设计师一个个人建议，最好不要过早的进行管理，前期还是要有专业上面的磨练和职场经历。不要为所谓的领导光环而放弃自己的初心，除非你真的在管理上面比较擅长和感兴趣，不然还是站在长远的角度来对自己的职业路径进行打磨。

 

3、平面物料设计基础认知

3.1、印刷品设计

平面设计中牵涉到的物料（印刷品）设计非常多，这里给大家简单梳理一些比较常见的类型，让大家有个大概的认知。

画册：一般为产品手册和企业宣传册等，起到宣传产品、品牌形象的目的。装订方式有骑马钉、蝴蝶钉，无线胶装、琐线胶装、精装。设计的时候页数能被 4 整除，骑马钉必要要是 4 的倍数，不然就会出现空白页，浪费成本。

单页/折页：这个在市场推广中非常常见，单页分为单面印刷和双面印刷，折页有对折页、三折页、四折页等等。印刷成本相对较低、分发比较方便，如果利用数码印刷，还能灵活的控制印刷数量，是企业推广中必不可少的印刷品之一。

海报：海报是比较常见的印刷品，较为常见的印刷材质有纸质、喷绘、写真、即时贴等。如果需要张贴一般背面带胶，如果是放进灯箱内的广告会采用背喷灯箱片。

包装：包装设计制作无论是材质、制作工艺等都非常多，按材料来分类比较常见的有纸质，铁质，木质，布质，皮质，亚克力，瓦楞包装、pvc 等。我们在设计的时候就要先确定好制作材质和工艺，这样才能精确的计算好产品容纳的空间。

展示架：在推广活动中是必备的形式之一，成本低廉、存放与携带方便都是其有利的优势。其中 X 展架和易拉宝是比较常见的，X 展架相对便宜，稳定性易拉宝相对好些。展示架常规尺寸：60cm*160cm 和 80cm*180cm。

横幅：无论是宣传还是组织活动都是较为常见的，大部分材料一般为锦纶牛津布。为了拍照的时候能够拉得更平整，也有一些会采用写真的形式，而且写真制作时间比较快。

其他：在工作中还有一些常见的印刷品，比如名片、台卡、优惠券、DM 单页、信封、信纸、广告伞等等。我们在接到印刷品设计需求的时候可以多问几家供应商，在了解好印刷材质和制作工艺之后再进行设计，这样会更加保证设计的还原度。

 

3.2、数码印刷与传统印刷的区别

数码印刷是近年来印前领域热门技术之一，是一体印刷，直接由电脑输出，在短时间内可以打印出来，但是传统印刷，是需要制版后才能完成印刷。

跟传统印刷相比，数码印刷速度快，成本低，可以一张起印，价格也不会太高，对于需求量不大的情况是最佳的选择。但是如果针对需求量大的批量印刷还是传统印刷比较优惠，因为中间有开机费，制版费，印刷费等等。

 

3.3、图文制作店能干嘛？

对于平面设计师来说印刷厂、数码打印店、图文制作店都是非常熟知的，图文制作店也是相对接触比较多的，很多印刷制作的小东西都是可以去里面得到解决。如名片、X 展架、易拉宝、横幅、台卡、亚克力字、灯箱广告等等都是可以去图文制作店完成的。

由于店面不是很大，所以一般需求量不大的印刷品比较适合。大部分的喷绘写真都是在里面制作的，如果遇到大型的喷绘（如舞台背景等）还是会去工厂完成。一般我们都是就近选择，也会适当的进行比价选择。作为公司层面还需要考虑一个因素就是费用结算方式，需要谈好一个合作方式，月结相对比较多。

二、设计

 

4、基本的美术功底

4.1、基本要素

平面设计除了给人视觉上的享受以外，更重要的是向消费者转达一种信息和理念。因此，平面设计不仅是视觉表层的美观度，更重要的是信息的传达。

平面设计主要由以下几个基本要素构成：

创意：是平面设计的第一要素，没有好的创意，就没有好的作品，创意中要考虑观众、传播媒体、文化背景三个条件。

构图：构图就是要解决图形、色彩和文字三者之间的空间关系，做到新颖，合理和统一。

色彩：好的平面设计作品在画面色彩的运用上注意调和、对比、平衡、节奏与韵律。

平面设计都是由这些要素通过巧妙的安排、配置、组合而成的。

4.2、专业术语

作为平面设计师需要将自己对信息的理解以图形、文字、色彩、排版等形式表达出来，形成独特的创意。我们除了能够满足需求以外，有时候对于一些专业术语的理解也是需要了解的，这样才能使设计作品更具理论依据。

以下列举部分常用的专业术语：

和谐：元素之间给我们的感觉和意识是否具备一种整体协调的关系；

对比：通过元素之间的大小、配色深浅等对比，使信息传达更高效，主体更加鲜明、作品更加活跃；

对称：通过左右图形完全相等形成对称图，中国传统图案、建筑、服饰等都具备对称性，构图相对稳定；

平衡：在平面设计中指的是根据图像的形量、大小、轻重、色彩和材质的分布作用与视觉判断上的平衡；

比例：部分与部分，或部分与全体之间的数量关系形成元素之间的比例，通过比例值来形成设计的节奏感；

重心：画面图像的轮廓变化，图形的聚散，色彩或明暗的分布都可对视觉重心产生影响，我们需要根据视觉协调来判断重心的偏移；

节奏：节奏是一种规律性的重复，用反复、对应等形式把各种变化因素加以组织，构成前后连贯的有序整体，即节奏；

韵律：画面中单纯的元素组合过于单调，通过元素的对比等形成节奏而产生犹如音乐的旋律感，成为韵律。

 

4.3、三大构成

所谓“构成”是一种造型概念，将不同形态的多个单元重组构成一个新的单元。三大构成指的是平面构成、色彩构成、立体构成。想要做好平面设计，好的美术基础打底是必须的，其次便是三大构成的学习，这是最基本的先决条件。

平面构成主要探索二维空间的视觉规律、形象的建立、骨格的组织、各种元素的构成规律等。

色彩构成是根据人们长期形成的色感而产生的一种思维定势，不同颜色的搭配，能够给人不同的心理感受，而色彩构成就是将这些思维定势总结出来，运用到后期的设计作品中。

立体构成是由二维平面形象进入三维立体空间的构成表现，用厚度来塑造形态、它是制作出来的。离不开材料、工艺、力学、美学，是艺术与科学相结合的体现。

4.4、透视基础

透视学即在平面上再现空间感、立体感，比较常见的有平行透视（又称一点透视，它只有一个消失点）、成角透视（二点透视）、倾斜透视（三点透视）等等。

透视基础无论在平面设计还是其他设计中也是非常重要的，我们在设计中会利用元素的大小、色彩的变化、造型的扭曲、虚实关系等来让画面具备一定的透视感觉，形成空间感。

4.5、造型能力

造型能力一般分为形象塑造能力和形象创作能力，通过观察能力、手眼协调能力、基础手绘能力、艺术表现能力等的不断训练可以提高形象塑造能力。这个也是我们学习基础美术的重要性，通过坚持写生来不断锻炼对物体视觉表层的还原度，能够快速准确的对一个形象进行还原。

形象创作能力需要设计师先具备形象塑造能力，在创作过程中对形象的完整性、识别性和艺术美感进行充分的掌握与再造，创作出全新的一个形象。这个对于设计师来说相对比较难，需要有一定的美术功底和设计功底，还需要大量的实战经验。

造型能力的掌握是对平面设计师最基本的要求，这样才能原创出更多的图形元素，形成富有创意的作品。

4.6、版式设计

板式设计是设计师所必备的基本功之一，通过设计师对信息进行视觉处理，在传达信息的同时，产生感官上的美感。比较常见的板式类型有骨格型、满版型、上下分割型、左右分割型、中轴型、曲线型、倾斜型、对称型、重心型、三角型、并置型、自由型和四角型等。

关于板式设计大家可以经常看一些杂志期刊，从中应该会学到很多排版技巧，这里也推荐大家几本关于板式设计的书籍，仅供参考：《进阶理解版式设计》、《设计的教室》、《平面设计中的网格系统》、《设计师的设计日记》等等。

元素与元素之间必然存在某种关联，我们在设计的时候需要找到这种关联因素。我们在排版的时候需要掌握的就是如何让信息与配图进行组合，来有效的传达信息的视觉流。

 

4.7、图形创意

图形创意研究的是图形语言，这个是作为设计师创造力的有力表现，通过思维发散不断地联想到元素之间的关联，通过艺术化处理，以视觉形式表现出来，达到吸引读者的目的。

简洁、生动、幽默是图形创意中最重要的一个原则，但是需要准确地传达信息主体，以达到识别性和直观性。图形创意比较常用的形式有联想、比拟法、同构图形（共生同构、异影同构、换置同构）、正负形、无理图形等等。

作为设计师脑洞的训练是非常重要的，都说创意来源于生活，但是很多设计师往往会不以为然。创意是对大脑中的碎片化映像进行二次重组，如果你没有所见所得所感，那么注定脑海是一片空白。

4.8、商业插画

插画设计现在也越来越受到重视，具备一定的手绘功底对于设计师是非常加分的。作为平面设计会经常需要在画面中进行商业插画或者图形的设计，一些刚入职场的设计师总是会选择以素材来替代。但是，由于版权意识也在逐步受到重视，素材的滥用可能会给公司和产品带来不必要的麻烦，以后对于设计师的手绘功底、专业执行力、创造力等会越来越重视。

插画以后会大量地运用到出版物、商业宣传、装饰画、影视多媒体、游戏设计和自由创作类等场景，在平面设计领域，我们接触最多的是文学插图与商业插画，这些都是需要设计师具备更好的美术功底和手绘能力。因此，你需要动起手来啦！

手绘的提升需要多练，即使你知道软件操作技巧和表现形式，但是对于画面的掌控和熟练度依然是需要反复磨练的。要给自己设定一些由浅入深的目标，比如从简单的线性插画开始到后面的肌理纹理的运用，从临摹开始到半原创，再到原创。手绘并非是科班生的专宠，动起手来你依然可以。

5、设计前的场景分析

在设计项目之前的场景分析是非常重要的，特别是线下活动相关的物料设计，如果设计上面有误差而不能与现场条件进行匹配，那么容易使得整体效果有所差别。

案例 01：在设计活动背景板的时候，除了精确测量现场制作尺寸以外，也要考虑使用场景。如果是舞台背景使用，就要考虑主持人、嘉宾、演员等的人物高度值，因为他们会遮挡住画面中关键信息的呈现。因此，我们在设计的时候就需要考虑这个因素而绕开重要信息的布局位置。

案例 02：比如一个舞台以展架作为背景画面的展示形式，那么我们设计的时候就不能只是以展架搭建出来的尺寸为准。通常会比测量尺寸大一点，以免出现白色画布的露出，影响整体画面的效果。

案例 03：我们在设计印刷物料的时候，需要考虑裁切的误差，毕竟大批量的印刷不可能是一张张裁切的。也就是我们所谓的出血线（让印刷画面超出的那条线），出血线外面的位置就是要被裁掉的地方。

案例 04：如果是一场户外的活动需要展架，我们在设计的时候需要确定好展架的材质。因为户外是有风的影响这个因素的，如果材质很轻就无法抵挡，需要采用能够固定在地面或者重量足够抵挡一般横风的材质。

案例就不一一列举，希望通过简单的几个案例，让大家知道设计前的场景分析在设计中的重要程度，让你的每一个设计都符合真实场景的设计需求。

 

6、灵感积累的重要性

无论做什么工作，职业习惯是非常重要的，作为设计师来说，养成良好的设计职场习惯也是至关重要。所谓的灵感积累就是日常对于优秀设计作品的采集与分类能力，去吸收优秀的设计作品，分析与发现作品背后的设计轨迹，这样才能不断地武装自己的创意脑海。

并不是接到项目，需要灵感的时候才去寻找，而是养成日常灵感积累与分类的习惯。虽然做平面设计，但是你的关注点并非仅仅如此，全面的设计认知将会使得你的脑洞更加灵活，不断地创作出更好的作品。

7、素材的利与弊

对于很多初入职场的设计师，会过于依赖素材的运用，虽然素材可以变相地提高工作效率，但是却会使得设计师养成一个过于依赖的习惯。素材的运用会降低设计师的设计动手能力，一旦遇到要求较高的项目就会难以执行。

所以，在面对素材上面通常都是利弊交加的，我们需要谨慎对待，千万不要过于依赖而降低自己的动手能力，要具备自己创造素材的能力，对于看到的视觉效果要能知道实现的原理和运用的技法。

 

8、日常自创素材的重要性

面对前面一个素材利弊的问题，我们引出关于日常自创素材的话题，虽然对于很多初入职场的设计师来说，自创素材需要更多的时间和精力，也不能很好地达到想要的效果。但是这个创作素材的过程不仅可以锻炼对于软件技法层面的熟练度，也能养成动手设计的习惯。

通过自己原创出来的素材不仅用得放心（无需考虑版权问题），也能积累属于自己的素材库，这将会使你进阶设计更快一步。

三、对接

 

9、印刷前

设计稿做得再好如果印刷出现问题，也是等于无用功一场。在提交印刷之前我们需要经过一轮排查，因为印刷一旦开始，就算中途停止也是会浪费很多成本（财务成本、时间成本等）。

提交印刷前我们需要注意哪些事项：

a. 印刷成品尺寸与设计尺寸比对，需要考虑好出血大小，不管什么印刷都得做出血，保证 3mm 以上；

b. 色彩模式的排查，大批量的印刷需要 cmyk 模式，虽然 rgb 打样是漂亮，但印刷出来就差了。如果是少量的数码印刷相对比较好些，提交印刷稿之前最好和印刷公司沟通一下；

c. 印刷文件排查三部曲，首先是文件是否存在隐藏元素；其次是元素是否有被锁定；最后是文字是否转曲，转曲文字是否有出现封闭部分被填充等；

d. 运用的设计软件不同，对于图片的嵌入方式也不同，我们要根据自己运用的软件进行检查，看看图片是否都嵌入到设计文件中，如果没有嵌入可能会出现图片丢失的情况；

e. 主意软件版本问题，作为设计师可能习惯性地经常更新最新版的设计软件，但是对于一些印刷厂来说可能还处于老版本状态，提交文件的时候尽量选择相对较低的版本格式，以防对方打不开或者打开有文件丢失的情况；

f. 最后的信息校对，在最后提交印刷文件前再次进行信息的全面检查，以防制作印刷文件的时候造成的信息丢失和元素错位等，也是对信息错误排查的重要一环。

在提交印刷文件前尽量多进行文件排查，以防一些小问题而导致制版费用和时间成本的浪费。

10、印刷中

对比印刷前来说，印刷过程中相对会比较好把控一点，只是中间会有一次看样的过程，这个过程是至关重要的，其余就是关于印刷周期的跟进与反馈。

在进行看样的时候，最好要求印刷厂进行实样出样，这样与真实的成品最为接近。现在很多的印刷厂为了节省成本采用数码打样，虽然不影响信息的校对，但是色差还是会存在的。关于色彩的还原也是很多大公司比较在意的，通常相差一点点都是需要反复出样的，所以千万不要草草了之。

在工作中，拿到实样之后，先自己对信息板式等进行走查，没有问题之后再提交给领导进行最终确认，一般是需要领导签字确认之后才进入大批量印刷，但是给他们签字之前作为设计师是要先签字确认的，这也是责任制落实。

进入印刷以后要明确交付的时间节点，有时候一些印刷厂为了节省成本，会故意延迟一点时间找到更多的客户进行拼版印刷。一些比较靠谱的印刷厂有时候也会提出这个需求，这样大家可以平摊一下成本，也能给公司节省一点开支，但是前提是不能影响质量和交付日期。

11、交付后

到达印刷交付这个环节基本整个印刷跟进也算完成了，只差最后的验收环节了。验收也是非常重要的，毕竟花了钱肯定要得到应有的质量和服务。不过对于现在很多公司来说，服务质量都在不断地上升，毕竟竞争环境还是很大的，没有好的服务和质量肯定会越来越丢失客户圈。

我们在验收的时候需要抽样检测，不要都在一个包装里面抽取，要多分几个不一样的包装进行抽样，这样才能全面的进行质量验收。除了检查是否有运输损坏以外，也要对内容、色彩等进行大致的排查，做好最后的关卡控制。

四、行业路径

12、杂事之下的代表作

对于大部分的平面设计师来说，在工作中遇到的杂事还是挺多的，设计类别也不会很精转。如果在这样的环境下你选择了应付工作，那么未来的职场将会不断地淡化你的存在。

无论是在何种规模的公司，都要主动去发现专业发挥的空间，我之前在一家仅有不到 10 人的小公司工作，杂事也比较多，做的东西也没办法形成体系。但是我利用业余时间把公司的全套设计都进行了翻新，牵涉到：公司品牌设计、官网设计、企业文化布置、一些基础的视觉定义等等。通过这样主动的工作热情，也很快随着公司的发展开始管理团队，最后也完成了公司多个 1.0 版本的产品设计和后期多个产品迭代，为后面的职场进阶之路铺下更好的基础。

在小公司你会学到更加全面的专业技巧，会使得你的专业能力更加综合，以后在进入大平台的时候不仅可以精转一个领域，也具备全面的专业能力，那么你的设计产出比将会更大。不要抱怨没有发挥空间和杂事太多，心态往往是主要的，杂事之下也能够形成一定的代表作，就看你是如何规划自己的职场路径和业余时间。

 

13、设计比赛带来的行业背书

个人还是比较支持设计师多参加比赛的，因为参加比赛不仅可以获得更多的设计经验，还能获得行业认可，加强行业背书。比起概念的练习，那么比赛会更加有针对性和主题性，产出的作品也算是满足一定的商业价值的。

作为平面设计师可以参加的比赛很多，比如 LOGO 设计，这个在很多征集网是很常见的，不过大家要选择一些稍微大型一点的，正规一点的单位，以免提交作品后没有任何反馈。还有很多公益广告设计大赛、吉祥物设计、插画设计、主题设计等等。

希望大家都可以利用比赛获得更多的设计作品和行业背书，不断地在设计职场路径上面获得更多的竞争力。

 

14、适合工作还是创业

其实很多的设计师都希望成为自由设计师，自主创业等等，创业或者自由职业看似简单轻松，其实需要考虑的方面比工作中需要考虑的多很多。工作中你可能完成自己的项目就有更多的自由时间自行分配，你也不用担心公司业绩和未来的发展计划，还有思考新一轮的盈利模式，寻找新的行业出口等等。

作为自由职业，你需要考虑客户的稳定性、专业能力的不断进阶、报价事项与尾款跟进、新客户的拓展等等。慢慢的你需要考虑成立工作室不断地发展规模，商业计划也是你需要不断思考创新的。所以结合自己的情况考虑，你到底适合工作还是创业？

如果初入职场的设计和经验较少的设计师不建议选择自由职业，你需要在工作中磨练实战经验与掌握专业以外的综合能力。比如没有稳定的客户资源将会很难维持生计；没有很强的自控能力将会无法充分利用好自己的时间，三天打鱼两天晒网一样是无法得到有力发展的；还需要不断地维护客户与开发市场，在专业上面也要不断突破与创新。

如果你觉得满足了以上一切的基本条件，那么再考虑开启你的创业之路吧！

五、对 UI 的影响

本文是给 UI 设计师大致的梳理一下平面设计相关的知识，现在有部分 UI 设计师也是从平面转过来的（包括我自己），无论是在软件层面还是视觉设计层面都具备一定的基础。图形设计能力对应到 UI 设计中的图标设计也是一大优势；板式设计基础会使得界面布局更加灵活多样；视觉上面的表现力可以使得对于色彩的把控更加到位，做到收放自如；品牌设计和插画设计等基础对于 UI 设计的深入来说就更为重要，这也是为什么很多平面设计转到 UI 设计会适应得更快的原因。只要把设计思维转变一下，多在交互逻辑和用户体验上面深入，就可以很快上手并得到不错的效果。

平面设计的基础知识会有助于 UI 设计在专业技法层面更加稳固，也能有助于我们应对职场中出现的更多问题，作为设计师需要更加综合的能力来帮助企业解决更多的设计问题。希望每一个 UI 设计师都能不断地提升平面设计能力，加大我们的专业深度。

小结

本文梳理了平面设计相关的知识体系，由于篇幅问题没办法全部展开细说，希望通过这个大纲的梳理，带给更多 UI 设计师一定的帮助。在未来的 UI 设计职场中，作为设计师要嘛就是往视觉设计方向延展，向全栈设计路线发展；也可以在细分领域深化，不过这个需要你在这一个板块有突出的表现，与大部分设计师形成差异化。

最后希望大家都能静下心来，专注打磨自己的专业，应对竞争力不断增大的设计职场，都能找到自己满意的团队一起努力，产出更好的作品。
